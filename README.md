Verificador de Número Primo

Sobre o programa
O objetivo é verificar se um número inteiro informado pelo usuário é **primo**.

Funcionalidades
- Validação de entrada (`int.TryParse`)
- Função `EhPrimo()` otimizada usando `Math.Sqrt()`
- Loop para repetir a operação

Como executar
1. Abra o terminal na pasta do projeto
2. Execute:
```bash
dotnet run
